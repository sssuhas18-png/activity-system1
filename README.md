# Activity System

A role-based full-stack backend system for managing student activities, approvals, and dynamic point allocation.

## 🚀 Features
- Role-based access (Admin / Student)
- Activity submission & approval workflow
- Dynamic points system
- Scalable backend architecture

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB

## ⚙️ Setup
```bash
npm install
npm start
